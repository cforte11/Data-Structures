# Linear Data Structures

Author: Christopher Forte
Date: March 14, 2026

## Overview

My directory contains personal implementations of eleven fundamental linear data structures. All structures are scripted across six programming languages, (C++, Python, JavaScript, TypeScript, Java, and Kotlin). Each is built from first principles with a consistent focus on correctness, efficiency, and educational clarity.
